Routing Security Step-by-Step Guide

Starting Point
This Step-By-Step Guide is intended to cover all aspects of getting started with routing security and maintaining the routing-security systems. This guide is a collection of documents and references to assist in installing, configuring, and maintaining the software that handles routing security.

This guide is focused on the rpki.net software. The rpki.net package is available on rpki.net.

It is assumed that the vast majority of rpki.net installations will be Relying Parties, with only a relatively small number of installations also acting as Certification Authorities. For this reason, the initial version of the Step-By-Step Guide focuses on the Relying-Party software. The Certification-authority software will be covered in due course.
